1. web开发中常用的资源分别是什么,具体作用是是什么,
   主要划分为两类资源:
	静态资源:HTML CSS JavaScript
	动态资源:JSP/Servlet

2.HTML标签_文本标签1
   文本标签,顾名思义就是html网页上嵌入文本内容,常用的文本标签
   注释标签,标题标签,换行标签,段落标签,分割线标签等标签

3.列表就是在网页上以一定的顺序显示内容,列表分为有序,无序,以及自定义列表.
<ol>        <!--有序-->
   <li></li>
   <li></li>
</ol>

<ul>        <!--无序-->
   <li></li>
   <li></li>
</l>


4.HTML标签_链接标签
    <a href="http://www.itcast.cn" target="_self">在本页面显示</a>
    <br>
    <a href="http://www.itcast.cn" target="_blank">条转到另一个页面</a>

5.HTML标签_块标签

	块标签就是咱们常用的div和span标签,大家一定理解块的含义,就是具有一定的范围和距离.

6.表格标签主要用途是网页的数据显得更加规整,其实就是用来布局网页的

7.HTML标签_表单标签_概述
什么是表单,表单的具体作用是什么,就是和服务器交互,向后台提交数据的。
表单向后台服务器提交数据,必须得有name属性。

8.表单中提交数据,是通过表单的输入项完成的,常见的输入项是input输入项
Type:text普通文本 
     password密码框 
     radio单选按钮 
     checkbox 复选框 
     file 文件输入项
     button 普通按钮
     submit 提交按钮
9.HTML标签_表单标签_表单项select&textarea
10. css是层叠样式表,目的是html网页的样式更加好看
    css的好处,使用css样式文件和html网页脱离,提高css样式文件的复用性
11.结合方式:四种结合方式 
	1.内部使用style属性 
	2.外部方式三种 
	* 使用style标签 
	* 使用link标签 
	* 使用@import属性导入

   css的扩展选择器并列,组合,以及扩展选择器
13.CSS是通过设置属性值,来改变html的样式,比如颜色属性,字体大小,边框粗细等属性等等.
